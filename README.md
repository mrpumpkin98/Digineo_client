## ⛏ 디기너 - 디깅모멘텀 유형검사 서비스 🗂

### ⛏ 디기너는?
> - 심리테스트를 시작하면, 다양한 질문을 통해 사용자의 성향과 흥미를 파악
> - 심리테스트 결과에 따라 사용자를 수집형, 몰입형, 컨셉형  3가지 유형으로 분류
> - 사용자의 유형에 따라 맞춤형 취미 추천
